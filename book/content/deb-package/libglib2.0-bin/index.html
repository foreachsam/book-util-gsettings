libglib2.0-bin 套件探索

下載套件

下載「libglib2.0-bin」這個套件。

$ apt-get download libglib2.0-bin

解開套件

解開「libglib2.0-bin_2.40.2-0ubuntu1_amd64.deb」

$ dpkg -x libglib2.0-bin_2.40.2-0ubuntu1_amd64.deb libglib2.0-bin

$ tree libglib2.0-bin
