libglib2.0-doc 套件探索

下載套件

下載「libglib2.0-doc」這個套件。

$ apt-get download libglib2.0-doc

解開套件

解開「libglib2.0-doc_2.40.2-0ubuntu1_all.deb」

$ dpkg -x libglib2.0-doc_2.40.2-0ubuntu1_all.deb libglib2.0-doc
